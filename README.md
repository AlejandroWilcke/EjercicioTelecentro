# EjercicioTelecentro
Una prueba sin frameworks.

# Instalación

El programa no tiene dependencias NPM, solo se necesita pullear el proyecto a la carpeta htdocs, con Apache corriendo en el puerto 80.

```
  git pull https://github.com/AlejandroWilcke/EjercicioTelecentro
```

Sino el proyecto está en linea: link.
